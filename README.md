## Cómo publicar en GitHub Pages (GRATIS, 5 minutos)

1. Ve a https://github.com y crea una cuenta gratuita (si no tienes)
2. Clic en **"New repository"** (botón verde)
3. Nombre: `fiesta7mo2026` — marca **Public** — clic **Create**
4. En la página del repo, clic **"uploading an existing file"**
5. Arrastra los 3 archivos (index.html, logo_7mo.png, README.md)
6. Clic **"Commit changes"**
7. Ve a **Settings → Pages → Source → Deploy from branch → main → / (root)**
8. Clic **Save**

En 2-3 minutos tu sitio estará en:
**https://TU-USUARIO.github.io/fiesta7mo2026**

## Uso

Al escanear el QR del recibo, el celular abre la página con los datos prellenados
y verifica automáticamente la autenticidad del recibo.

También se puede ingresar manualmente:
- Número de recibo (001, 002, etc.)
- Código completo (RC-2026-001-XXXXXXXX)
